Welcome to the Teleco Customer Churn Prediction App

This app helps you predict customer churn based on derived features like Engagement Score, Customer Frequency, and Long Term Stability. These are calculated using inputs such as Monthly Charges, Tenure, Tech Support availability, and Contract Type.

Key Features:

  • Engagement Score: Higher scores indicate lower probability of churn. However, a low engagement score combined with tech support and a valid contract still suggests a lower chance of churn. It acts as an early warning sign.
  • Customer Interaction Frequency: When Tech Support = Yes, value is 1 (indicating customer is actively using support). If Tech Support = No and Internet Service = No, value is 0 (no interaction expected).
  • Long Term Contract Stability: Value is 0 for Month-to-Month contracts (indicating instability), and 1 for One-Year or Two-Year contracts (indicating greater customer retention likelihood).

Prediction Capabilities:

  • Predict the likelihood of a customer churning using ML model
    Prediction: 1 = Customer Will Churn, 0 = Customer Will Not Churn
